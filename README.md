`Yshanka`: Yhat ScienseOps poorman's drop-in replacement
========================================================

Installation
----------------

```
$ git clone ... ; cd ...
$ conda env create -n yshanka
$ python manage.py db upgrade
$ python manage.py seed
```

Run
----------------

```
$ docker-machine start
$ python manage.py run
```
